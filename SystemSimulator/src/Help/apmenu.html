The Autopilot Menu

The Autopilot menu is the menu in the top left hand corner of the screen. This menu provides you with offline and online help as well as preferences for the Autopilot software.

About Autopilot, Visit Website & Visit Development Page

These are all pretty self explanatory.

Help

This is the help that you are reading now, so you obviously found out how to use it.

Preferences

These are the options you can choose to change the way Autopilot works for you. For more information, go to the Preferences page.

Exit

Also rather self explanatory. It closes the whole of the Autopilot process.
