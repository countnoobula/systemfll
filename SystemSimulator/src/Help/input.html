The Input Window

The input window is the window that allows you to choose exactly what you would like the robot to do and looks like the example below. You can add all your robot details into this window and it will work out all the types of motor constraints for you. You can also choose exactly how far you want the robot to move in rotations.

This window is not yet fully functional and not all of the features are available yet. Please check for updates to receive it when it comes out.
